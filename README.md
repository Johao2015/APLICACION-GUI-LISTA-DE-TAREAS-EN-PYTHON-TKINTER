# Aplicación GUI Lista de Tareas

**Autor:** Johao Flores

## Descripción
Esta es una aplicación de escritorio desarrollada en Python usando Tkinter que permite gestionar una lista de tareas.

## Funcionalidades
- Añadir tareas
- Marcar tareas como completadas
- Eliminar tareas
- Añadir tareas presionando ENTER
- Doble clic para completar tareas

## Requisitos
- Python 3.x

## Cómo ejecutar
1. Abrir el proyecto en PyCharm o cualquier IDE
2. Ejecutar el archivo `todo_app.py`

## Uso
- Escribe una tarea en el campo de texto
- Presiona "Añadir Tarea" o ENTER
- Selecciona una tarea y presiona:
  - "Marcar como Completada"
  - "Eliminar Tarea"

## Notas
El estado completado se muestra con un ✔ delante de la tarea.
